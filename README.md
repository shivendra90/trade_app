# Breeze API Automated Trading App

This is my trading application developed with Claude and Codex 5.3.
Essentially includes the live trading app `live_trade_bot.py` that has both live as well as dummy paper trading modes.
Also has the backtester module that successfully backtests strategies namely RSI Mean Reversion, Breakout and SMA crossover with automatic CSV trade tracker and a visual representation of the specified day's
